SharpFly is a FlyFF v15 server emulator written from scratch in C#.<br>
<br>
This project was created for learning purposes.<br>
<br>
<br>
Commit message prefixes:<br>
<br>
LS: Login server<br>
WS: World server<br>
PL: Packet library<br>
UL: Utility library<br>
<br>
